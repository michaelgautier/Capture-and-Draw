Detailed Description

Types for the interaction module.

Description
Primary formats for communication between a regular program and the interaction module.
Remarks
Began as struct cartesoid and evolved into the following types. This is the most important part of the solution as it determines the how the program is organized.
Precondition
ISO C++ 11 or later compliant compiler with an available C++ 11 STL implementation.
